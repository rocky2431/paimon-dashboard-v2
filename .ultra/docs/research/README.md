# Research Documents

This directory contains research outputs from `/ultra-research` sessions.

## Naming Convention

`{date}-{topic}.md`

Example: `2025-12-15-ui-component-library-selection.md`

## Contents

Research documents typically include:

1. **Problem Statement** - What we're researching
2. **Options Analyzed** - Alternatives considered
3. **Six-Dimensional Analysis** - Evaluation matrix
4. **Recommendation** - Final decision with rationale
5. **Action Items** - Next steps
